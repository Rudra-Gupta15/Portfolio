## Features

- 🤖 Advanced AI/ML model integrations
- 👁️ Real-time Computer Vision demos
- 🎮 Interactive Game Development showcases
- 📱 Responsive React-based UI/UX
- ⚡ High-performance animations and transitions
- 📊 Deep Learning project visualizations
- 📧 Integrated contact and networking modules

## Tech Stack

- React.js
- Tailwind CSS / Framer Motion
- Python (TensorFlow, PyTorch, OpenCV)
- JavaScript / Three.js
- Google Fonts (Cormorant Garamond, Jost, DM Mono)

## Deployment

This site is deployed on Vercel. Any push to the main branch will automatically deploy.

## Local Development

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to view the site locally at `localhost:3000`
