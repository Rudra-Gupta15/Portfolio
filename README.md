- **👁️ Computer Vision**: Real-time ASL recognition using YOLOv8 and multi-scale CNNs (RGB/HSV/Grayscale).
- **🎨 Premium UI/UX**: Custom Glassmorphism design system built with Vanilla CSS and React 18.

## 🚀 Projects & Extensions

### 🌐 [BalanceTab — Gamer Dashboard](https://chromewebstore.google.com/detail/balancetab-%E2%80%94-gamer-+-offi/nglnanlbnedkffjgncmokibcliabkpki)
A Chrome New Tab replacement fusing gamer aesthetics with productivity. Features 7 built-in HTML5 Canvas games, real-time weather, and an AI quick-launch dock.

### 🧠 [ResumeAI — Premium ATS Analyzer](https://resume-ai-analyzer-coral.vercel.app/)
An AI-powered ATS resume analyzer delivering instant scoring and domain-specific feedback. Supports both Cloud (Groq) and fully Offline (Ollama) modes.

### 🔖 [TimeMark — Video Bookmarks](https://chromewebstore.google.com/detail/timemark-%E2%80%94-video-timestam/kdpmjbeocligojphcnadcjobbpbkkbea)
A high-utility Chrome extension for YouTube, allowing users to bookmark exact timestamps with a native in-player button and green-flash feedback.

### 🎮 [Snow Run Game](https://snow-run-game.vercel.app/)
A multi-level racing game built purely with React. Includes physics-based movement, obstacle avoidance, and boss fights with zero game engine dependencies.

## 🛠️ Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rudra-Gupta15/rudra-portfolio.git
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Start development server**
   ```bash
   npm start
   ```
